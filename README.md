# RDLC Report Tester v1.01

Preview RDLC reports with BC XML data — without deploying to Business Central.

## How it works
Drop .rdlc and .xml files → get PDF preview instantly.

## Install
Download RdlcReportTester-Setup-1.01.exe from Releases.
Requires WebView2 Runtime (installer will prompt if missing).

## Tech stack
- ReportViewerCore.NETCore — RDLC rendering
- Microsoft.Web.WebView2 — PDF preview
- .NET 8 / WPF
